# Notebooks

Exploratory notebooks for RF-Sentinel research.

## Contents

| Notebook | Description |
|----------|-------------|
| `01_dataset_exploration.ipynb` | Explore the RadioML 2018.01A dataset structure, distributions, and sample waveforms |

## Usage

```bash
pip install jupyter
jupyter notebook notebooks/
```

> **Note**: Notebooks require the RadioML dataset to be available at the configured path.
